# Vue-ssr-express

## Project setup
```
npm install
```
## Project update
```
npm update
```
### JSON server start (http://localhost:3000)
```
json-server --watch db.json
```

### Compiles and hot-reloads for development (http://localhost:5000)
```
npm run dev
```

### Compiles and minifies for production (http://localhost:5000)
```
npm run build
```
