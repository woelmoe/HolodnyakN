# Vue-ssr-express

## Project setup
```
npm install
```
## Project update
```
npm update
```

## Animate.css setup
```
npm install animate.css --save
```

### JSON server start
```
json-server --watch db.json

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
